Web Development Projects

Project 1: Netflix Clone

This project is a clone of Netflix built using HTML, CSS, and JavaScript. It replicates the user interface and functionality of the popular streaming platform, allowing users to browse through movies and TV shows, view details, and play trailers.

Project 2: Game of Stone Paper Scissors

This project is a web-based game of Stone Paper Scissors implemented using HTML, CSS, and JavaScript. It provides an interactive interface where users can play the classic game against the computer, with options to choose between stone, paper, or scissors.

Project 3: Blogging Website

This project is a simple blogging website created using HTML, CSS, and JavaScript. It allows users to create, edit, and delete blog posts, as well as interact with other users through comments and likes. The website features a clean and responsive design for optimal user experience.

Project 4: Spotify Clone

This project is a clone of the popular music streaming service Spotify, developed using HTML and CSS. It mimics the design and layout of the Spotify web application, allowing users to browse through music tracks, create playlists, and enjoy their favorite songs.
